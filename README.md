# pdf-test

Building a PDF page splitter using [vue-pdf](https://github.com/TaTo30/vue-pdf) and [vueUse](https://github.com/vueuse/vueuse).

![](./pdf-split.gif)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
